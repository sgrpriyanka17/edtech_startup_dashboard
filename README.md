**1.Ed Tech Online Courses Insights Dashboard**

A dynamic,interactive data visualisation tool built to explore around 10,000 online courses from popular online learning platforms like Coursera, Udacity, Simplilearn, and FutureLearn including information about their course type, language, skills, instructor, rating, duration and number of viewers.

**2.Description**

This dashboard transforms raw, multi-dimensional educational data into actionable insights through intuitive visual storytelling, enabling users to understand trends through platform comparison.Users can evaluate how different providers perform across key metrics.The dashboard helps in evaluating course effectiveness by correlating multiple indicators such as ratings, enrollments, and duration.

**3. Tech Stack**

The dashboard was built using the following tools and techniques:

_Power BI Desktop_- Main data visualisation platform used for report creation. 

_Power Query_ Data transformation and cleaning layer for reshaping and preparing the data. 

_DAX(Data Analysis Expressions)_- Used for calculated measures, dynamic visuals, and continual logic.

_Data Modeling _Relationship established among tables to enable cross-filtering and aggregation. 

_File Format_-.pbix for development and .png fro dashboard previews. 

**4.Data Source**- https://www.kaggle.com/datasets/khaledatef1/online-courses

**5.Problem Statement**

With the rapid growth of online learning platforms such as Coursera, Udacity, Simplilearn, and FutureLearn, a vast number of courses are available across diverse domains, skill levels, and languages. However, this abundance of information creates challenges for learners and stakeholders in effectively identifying relevant, high-quality, and suitable courses.
